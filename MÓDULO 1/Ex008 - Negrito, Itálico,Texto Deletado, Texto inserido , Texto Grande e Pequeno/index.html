Principais Formatações

Negrito / Destaque

Nessa frase termo Termo em negrito usando a tag b (Não semântica)

Nessa frase termo Termo em destaque usando a tag strong (Semântica)

Usamos negrito para destacar palavras importantes e destaque para enfatizar o significado do texto.

Itálico / Cursiva

Nessa frase termo Termo em itálico usando a tag i (Não semântica)

Nessa frase termo Termo em ênfase usando a tag em (Semântica)

Usamos itálico para destacar palavras estrangeiras ou títulos de obras e ênfase para dar ênfase ao significado do texto.

Texto Marcado

Nessa frase termo Termo em destaque que está marcado usando a tag mark (Semântica)

Usamos texto marcado para destacar informações importantes ou relevantes dentro do texto.

Texto Grande / Pequeno

Estamos criando um texto grande e um texto pequeno

Usamos texto grande Não mais está viavel usar, apenas aumentar a letra pelo CSS texto pequeno para indicar informações complementares.

Texto deletado

Podemos marcar um texto como exlucido para indicar que ele deve ser lido, mas não considerado

Texto inserido

Podemos marcar um texto como inserido para indicar que ele foi adicionado ao documento

Existe também os sublinhados com a tag U (Não semântica)
